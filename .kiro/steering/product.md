# Product Overview

Pac-Kiro is a classic Pac-Man style arcade game featuring the Kiro logo as the player character. Players navigate a maze collecting dots while avoiding (or eating) ghosts.

## Core Features
- Grid-based movement with arrow key controls
- Classic Pac-Man maze layout (28x31 tiles)
- 4 AI-controlled ghosts with chase behavior
- Power pellets that allow eating ghosts temporarily
- Lives system (start with 3 lives)
- High score tracking via backend API
- Smooth frame-based movement timing

## Scoring
- Regular dots: 10 points
- Power pellets: 50 points
- Eating ghosts: 200 points each

## Game States
- Start screen with instructions
- Active gameplay
- Level complete (all dots collected)
- Game over (all lives lost)
